# SportPlay Repository

## Installation Kodi

1. **Source** : Paramètres → Gestionnaire de fichiers → Ajouter une source
   - URL : `https://m4t-r.github.io/Sportplay/`
   - Nom : `SportPlay`

2. **Repo** : Addons → Installer depuis un zip → SportPlay → `zips` → `repository.sportplay` → installer

3. **Addon** : Addons → Installer depuis un repository → SportPlay Repository → Extensions vidéo → SportPlay

## Mise à jour

```bash
python update.py /chemin/vers/plugin.video.sportplay-X.X.X.zip
git add -A && git commit -m "vX.X.X" && git push
```
